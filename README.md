# jQuery.cssEdit

## margin / paddingを操作しやすいツール。

### イベントセット
```javascript
$(function(){
    $('img').cssedit();
});
```

### 操作方法

動かす要素をダブルクリックします。

AWDSのどれかを押しっぱなしで、
特定方向のmargin/paddingを操作します。

上下のスクロールでpxを操作します。

A : Left
W : Top
D : Right
S : Botom

Shift + F : margin/paddingの切替
